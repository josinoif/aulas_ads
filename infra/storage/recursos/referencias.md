# Referências e leituras recomendadas

## Documentação oficial

- [MinIO — Documentação](https://min.io/docs/minio/linux/index.html) — Instalação, conceitos e API S3-compatível.
- [MinIO — JavaScript SDK](https://min.io/docs/minio/linux/developers/javascript/API.html) — Cliente Node.js para MinIO.
- [AWS S3 — Documentação](https://docs.aws.amazon.com/s3/) — Conceitos, operações e melhores práticas.
- [AWS SDK for JavaScript v3 — S3](https://docs.aws.amazon.com/sdk-for-javascript/v3/developer-guide/s3-examples.html) — Exemplos com S3 no Node.js.
- [LocalStack — S3](https://docs.localstack.cloud/user-guide/aws/s3/) — Como usar S3 emulado localmente.

## Ferramentas

- [Multer (Node.js)](https://github.com/expressjs/multer) — Middleware para upload de arquivos no Express.
- [Docker](https://docs.docker.com/) — Para rodar MinIO e LocalStack em containers.

## Material relacionado no repositório

- [Frontend React — Arquivos](../../frontend/reactjs/10-arquivos/) — Upload e download no navegador (FormData, File API).
- [Backend NestJS — Upload de arquivos](../../backend/nest/6.upload_arquivos.md) — Upload em disco com NestJS e Multer.
