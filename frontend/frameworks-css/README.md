# Frameworks CSS — Fundamentação e Prática para o Mercado

**Público:** Ensino superior (Desenvolvimento de Software)  
**Objetivo:** Fundamentação teórica, pensamento crítico e aplicação prática de frameworks CSS, com foco em decisão técnica e problemas reais do mercado.

---

## O que você vai levar dessa aula

- **Fundamentação:** O que são frameworks CSS, por que existem e quando fazem (ou não) sentido.
- **Visão de mercado:** Conhecer os frameworks mais usados na indústria, com prós, contras e casos de uso reais.
- **Decisão técnica:** Critérios para escolher (ou não) um framework e como argumentar sua escolha em equipe e em entrevistas.
- **Mãos na massa:** Desafios que simulam problemas reais (dashboard, landing page, migração, protótipo sob pressão) usando diferentes ferramentas.

A ideia não é “decorar” um framework, e sim **saber quando e como usá-los** — e quando abrir o código e escrever CSS puro ou outra solução.

## Como usar

1. **Leia na ordem** — a fundamentação sustenta a comparação e a decisão; os desafios aplicam tudo isso.
2. **Faça os desafios de verdade** — escolha pelo menos um framework (ex.: Bootstrap ou Tailwind), monte o ambiente e implemente.
3. **Discuta em grupo** — “por que esse framework aqui?” e “o que mudaria no projeto X?” são perguntas típicas de processo seletivo e de planejamento de produto.
4. **Consulte a seção de referências** ao final de cada arquivo quando quiser aprofundar.
