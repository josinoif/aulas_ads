# Referências e leituras recomendadas

## Documentação oficial

- [React — Documentação](https://react.dev/) — Conceitos, hooks, guias e referência da API.
- [React Router](https://reactrouter.com/) — Documentação do React Router (roteamento no frontend).

## Ferramentas e ecossistema

- [Vite](https://vitejs.dev/) — Ferramenta recomendada para criar e rodar projetos React (rápida, recarga instantânea). Use `npm create vite@latest meu-app -- --template react`.
- [npm](https://www.npmjs.com/) — Registro de pacotes; usado com `npm install` e `npx`.

## APIs e HTTP

- [MDN — Fetch API](https://developer.mozilla.org/pt-BR/docs/Web/API/Fetch_API) — Requisições HTTP nativas do navegador.
- [Axios](https://axios-http.com/) — Cliente HTTP para JavaScript/React.

## Estilo e layout

- [MDN — Flexbox](https://developer.mozilla.org/pt-BR/docs/Web/CSS/CSS_Flexible_Box_Layout) — Layout em uma dimensão.
- [MDN — Grid](https://developer.mozilla.org/pt-BR/docs/Web/CSS/CSS_Grid_Layout) — Layout em grade.

## Bibliotecas citadas no curso

- [Material-UI (MUI)](https://mui.com/)
- [Chakra UI](https://chakra-ui.com/)
- [React Hook Form](https://react-hook-form.com/)
- [Zustand](https://github.com/pmndrs/zustand)
- [TanStack Query (React Query)](https://tanstack.com/query/latest)

## Prática

- [CRUDCrud](https://crudcrud.com/) — API REST gratuita para exercícios de CRUD (formulários e listagens).
