# Desenvolvimento de layouts para páginas web e mobile

**Público:** Ensino superior (Desenvolvimento de Software)  
**Objetivo:** Fundamentação teórica e prática de layouts responsivos, com foco em **Flexbox**, **CSS Grid**, usabilidade e tomada de decisão — usando CSS puro e frameworks quando fizer sentido.

---

## O que você vai levar dessa aula

- **Fundamentação:** O que é layout na web e no mobile, por que estrutura e consistência importam e como isso se conecta ao mercado (entregas, manutenção, trabalho em equipe).
- **Ferramentas do CSS:** Flexbox e CSS Grid em profundidade — conceitos, quando usar cada um e como combiná-los para resolver problemas reais.
- **Usabilidade:** Hierarquia visual, espaçamento, alvos de toque, responsividade e acessibilidade; como implementar isso com os recursos de layout.
- **Decisão técnica:** Quando usar Flexbox, quando usar Grid, quando apoiar-se em um framework (Bootstrap, Tailwind) e quando escrever CSS puro.
- **Mãos na massa:** Desafios e exercícios de fixação que simulam problemas reais (dashboard, formulário, card, página responsiva).

## Como usar

1. **Siga a ordem** — A fundamentação sustenta Flexbox/Grid; a usabilidade mostra o “porquê” das escolhas; a decisão e os desafios aplicam tudo.
2. **Pratique com CSS puro primeiro** — Entenda Flexbox e Grid nativos antes de depender só de classes de framework.
3. **Faça os exercícios de fixação** após cada bloco conceitual; use os desafios para integrar layout + usabilidade + decisão.
4. **Discuta em grupo** — “Por que Flexbox aqui e Grid ali?” e “Como garantir alvos de toque em mobile?” são perguntas típicas de entrevista e de revisão de código.
